# Components

Browse and learn about all available Harukit components.

## Overview

Harukit provides a collection of beautiful, accessible UI components built on top of Radix UI primitives. All components are:

- **Fully customizable** - Edit the code directly in your project
- **Accessible** - Built with WAI-ARIA compliance
- **TypeScript ready** - Full type safety out of the box
- **Tailwind styled** - Styled with Tailwind CSS and CSS variables

## Available Components

### Layout Components

#### Accordion

Collapsible content sections with smooth animations.

```bash
npx harukit@latest add accordion
```

**Dependencies:** `@radix-ui/react-accordion`

**Usage:**

```tsx
import { Accordion, AccordionContent, AccordionItem, AccordionTrigger } from "@/components/accordion"

export default function MyAccordion() {
  return (
    <Accordion type="single" collapsible>
      <AccordionItem value="item-1">
        <AccordionTrigger>Is it accessible?</AccordionTrigger>
        <AccordionContent>
          Yes. It adheres to the WAI-ARIA design pattern.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  )
}
```

#### Card

Container for content with header, content, and footer sections.

```bash
npx harukit@latest add card
```

**Dependencies:** None

**Usage:**

```tsx
import { Card, CardContent, CardDescription, CardHeader, CardTitle } from "@/components/card"

export default function MyCard() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description goes here</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Card content goes here</p>
      </CardContent>
    </Card>
  )
}
```

### Form Components

#### Button

Versatile button with multiple variants and sizes.

```bash
npx harukit@latest add button
```

**Dependencies:** `@radix-ui/react-slot`, `class-variance-authority`

**Usage:**

```tsx
import { Button } from "@/components/button"

export default function MyButtons() {
  return (
    <div className="space-x-2">
      <Button>Default Button</Button>
      <Button variant="destructive">Delete</Button>
      <Button variant="outline">Outline</Button>
      <Button size="sm">Small</Button>
    </div>
  )
}
```

**Variants:**
- `default` - Primary button style
- `destructive` - Red button for destructive actions
- `outline` - Outlined button style
- `secondary` - Secondary button style
- `ghost` - Ghost button style
- `link` - Link-style button

**Sizes:**
- `default` - Default size
- `sm` - Small size
- `lg` - Large size
- `icon` - Square button for icons

#### Input

Form input field with proper styling and focus states.

```bash
npx harukit@latest add input
```

**Dependencies:** None

**Usage:**

```tsx
import { Input } from "@/components/input"

export default function MyInput() {
  return (
    <div className="space-y-2">
      <Input type="email" placeholder="Email" />
      <Input type="password" placeholder="Password" />
      <Input type="text" placeholder="Username" />
    </div>
  )
}
```

#### Label

Form label with accessibility features.

```bash
npx harukit@latest add label
```

**Dependencies:** `@radix-ui/react-label`

**Usage:**

```tsx
import { Label } from "@/components/label"
import { Input } from "@/components/input"

export default function MyForm() {
  return (
    <div className="space-y-2">
      <Label htmlFor="email">Email</Label>
      <Input id="email" type="email" placeholder="Enter your email" />
    </div>
  )
}
```

### Feedback Components

#### Tooltip

Hover tooltips with customizable content.

```bash
npx harukit@latest add tooltip
```

**Dependencies:** `@radix-ui/react-tooltip`

**Usage:**

```tsx
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from "@/components/tooltip"
import { Button } from "@/components/button"

export default function MyTooltip() {
  return (
    <TooltipProvider>
      <Tooltip>
        <TooltipTrigger asChild>
          <Button variant="outline">Hover me</Button>
        </TooltipTrigger>
        <TooltipContent>
          <p>This is a tooltip</p>
        </TooltipContent>
      </Tooltip>
    </TooltipProvider>
  )
}
```

| Category | Components | Description |
|----------|------------|-------------|
| **Layout** | Accordion, Card | Components for structuring content |
| **Form** | Button, Input, Label | Components for user input and interaction |
| **Feedback** | Tooltip | Components for user feedback |

## Adding Components

To add components to your project:

```bash
# Add a single component
npx harukit@latest add button

# Add multiple components
npx harukit@latest add button card input

# Add all components
npx harukit@latest add accordion button card input label tooltip
```

## Customization

All components are copied to your project, so you can:

- Modify the styling by editing the component files
- Add new variants or props
- Customize the behavior
- Integrate with your design system

## Dependencies

Components automatically install their required dependencies:

- **Radix UI primitives** - For accessibility and behavior
- **class-variance-authority** - For variant management
- **clsx** - For conditional class names
- **tailwind-merge** - For Tailwind class merging
